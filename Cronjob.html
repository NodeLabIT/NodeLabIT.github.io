Cronjob

Manage cronjobs that are executed during system-runtime

Methoden

registerJob(time, handle)

Define a callback that is executed every time the cronjob runs. The time at which the cronjob is executed has to be determined.
Parameters:
Name Type Description
time string The time/period, as cron-syntax, at which the cronjob should run.
handle Cronjob~cronjobHandler The callback that is executed every time the cronjob runs.

Type Definitions

cronjobHandler()

Callback as function to execute every time the cronjob runs.
